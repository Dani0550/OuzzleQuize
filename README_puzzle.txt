Se trata de un proyecto en el trabajo con la API de drag and drop, donde pongo en práctica mis hailidades en JS y CSS trabajando en una lógica que se centera en el funcionamiento básico de un juego de puzzle. 

Támbien con distintas de condicionales y funciones algo más complejas para evitar trampas y dando un sentido al juego donde tiene que completar los 6 puzzles para terminar y acceder al apartado de scorre, donde podrás ver tu posicionamiento en la tabla en competencia con personajes ficticios para poder entender que tan bien ha sido tu partida. 

Mediante este sistema de scoore también muestro como es posible evaluar y puntuar al jugador en referencia al tiempo que ha tardado en hacer los puzzle. El sistema de scorre puntua cada puzle que se realiza por separado para al final terminar sumadno la puntuación y obtener tu posición en la tabla.

Trabajo con posicionamiento de las imagenes, lo que dificulta mucho realizar los puzles, ya que el posicionamiento se adapta a la casilla en la que se le desplaza, por ende la vision de la imagen se ve alterada.

//////////////////////////////////////////////////////////////////////////////////////////////////////////

This is a project working with the drag and drop API, where I put into practice my skills in JS and CSS working on a logic that focuses on the basic functioning of a puzzle game.

Also with different conditional and more complex functions to avoid cheating and giving a sense of the game where you have to complete the 6 puzzles to finish and access the scorre section, where you can see your position in the table in competition with fictitious characters to understand how well your game has been. 

By means of this scorre system I also show how it is possible to evaluate and score the player with reference to the time he/she has taken to complete the puzzles. The scorre system scores each puzzle that is done separately to end up adding up the score and get your position in the table.

I work with image positioning, which makes it very difficult to carry out the puzzles, as the positioning adapts to the box in which it is moved, and therefore the vision of the image is altered.

////HOW TO USE

- Download the zip file
- Extract the folder and move it to where you feel comfortable.
- Once you have done the above steps just run the main .html file.
